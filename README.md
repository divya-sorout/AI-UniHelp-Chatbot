# AI UniHelp Chatbot 🎓🤖

AI UniHelp is a smart AI-powered university assistant chatbot developed to simplify student support and improve access to university-related information through an interactive and responsive web platform.

The chatbot is designed to provide instant answers to common student queries including admissions, courses, fee structure, placements, campus facilities, academic support, events, and general university information. This project demonstrates the practical implementation of AI-driven assistance in the education sector using modern web technologies.

Built with a strong focus on user experience, responsiveness, scalability, and real-world usability, AI UniHelp aims to create a seamless digital support system for students.

## ⚙️ Installation & Setup

### 1️⃣ Clone Repository

```bash
git clone https://github.com/divya-sorout/AI-UniHelp-Chatbot.git
```

---

### 2️⃣ Navigate to Project Folder

```bash
cd AI-UniHelp-Chatbot
```

---

### 3️⃣ Install Dependencies

```bash
npm install
```

---

### 4️⃣ Start Development Server

```bash
npm start
```

The application will run locally at:

```bash
http://localhost:3000
```

---

## 🔥 Firebase Deployment

### Build Project

```bash
npm run build
```

### Deploy to Firebase

```bash
firebase deploy
```
